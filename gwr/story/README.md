# GWR Story

This directory is for the long-form plain-language story of the `Gap Winner
Rule` (`GWR`) and the plots that support it.

The intended workflow here is:

1. draft the narrative skeleton,
2. identify the visual claims each section needs,
3. generate only the figures required to support those claims,
4. embed the figures into the story document.

## Current Files

- [`gwr_story_skeleton.md`](./gwr_story_skeleton.md): section-by-section
  narrative skeleton and plot plan.
- `plots/`: figure outputs for the story document.
